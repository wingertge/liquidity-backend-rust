juniper_warp

This repository contains the warp web server integration for Juniper, a GraphQL implementation for Rust.

Documentation

For documentation, including guides and examples, check out Juniper.

A basic usage example can also be found in the Api documentation.

Examples

Check examples/warp_server for example code of a working warp server with GraphQL handlers.
